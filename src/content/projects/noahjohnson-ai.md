This very site. A personal blog and portfolio designed to look and feel like the Claude Code desktop app — three-pane shell, sidebar of "sessions," warm dark palette, terracotta accent.

Built on **Astro 6** with **Tailwind CSS 4** and the **Geist** type family. Static-rendered, deployed on **Vercel**. Search via **Pagefind**. RSS + sitemap built in.

Source on GitHub: [github.com/nojoatx/noahjohnson-site](https://github.com/nojoatx/noahjohnson-site)
